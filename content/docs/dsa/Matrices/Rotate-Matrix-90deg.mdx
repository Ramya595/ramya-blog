## âŒ 1. Brute Force (Using Extra Matrix)

### **Idea**

* Create a new matrix
* Place elements according to rotated position:
  `res[col][n - 1 - row] = matrix[row][col]`

### **Code (JavaScript)**

```js
function rotate(matrix) {
  const n = matrix.length;
  const res = Array.from({ length: n }, () => Array(n).fill(0));

  for (let row = 0; row < n; row++) {
    for (let col = 0; col < n; col++) {
      res[col][n - 1 - row] = matrix[row][col];
    }
  }

  // Copy back into original matrix
  for (let i = 0; i < n; i++) {
    for (let j = 0; j < n; j++) {
      matrix[i][j] = res[i][j];
    }
  }
}
```

### Complexity

| Metric | Value                    |
| ------ | ------------------------ |
| Time   | **O(nÂ²)**                |
| Space  | **O(nÂ²)** (extra matrix) |

## ðŸš€ 2. Optimal In-Place Approach (Transpose + Reverse)

This is the classic optimal solution.

### **Idea**

The 90Â° rotation can be achieved by:

### Step 1: **Transpose**

Swap `matrix[i][j]` with `matrix[j][i]`.

### Step 2: **Reverse each row**

After these two operations, the matrix becomes rotated.

---

## ðŸ§‘â€ðŸ’» Code (JavaScript Optimal)

```js
function rotate(matrix) {
  const n = matrix.length;

  // Step 1: Transpose
  for (let i = 0; i < n; i++) {
    for (let j = i + 1; j < n; j++) {
      [matrix[i][j], matrix[j][i]] = [matrix[j][i], matrix[i][j]];
    }
  }

  // Step 2: Reverse each row
  for (let i = 0; i < n; i++) {
    matrix[i].reverse();
  }
}
```

# ðŸŒŸ Optional: In-Place Layer-by-Layer Rotation

This approach rotates **4 cells at once** in layers (rings of matrix).
More mathematical, less intuitive.

```js
function rotate(matrix) {
  const n = matrix.length;

  for (let layer = 0; layer < n / 2; layer++) {
    let start = layer;
    let end = n - 1 - layer;

    for (let i = 0; i < end - start; i++) {
      let top = matrix[start][start + i];

      matrix[start][start + i] = matrix[end - i][start];
      matrix[end - i][start] = matrix[end][end - i];
      matrix[end][end - i] = matrix[start + i][end];
      matrix[start + i][end] = top;
    }
  }
}
```

### Complexity

* **Time:** O(nÂ²)
* **Space:** O(1)
* Works purely layer-by-layer
