## ğŸ¢ 2. Brute Force Approach (Linear Search)

The simplest way is to scan every element from left to right.

### **Idea**

Check each element one by one until you find the target.

### **JavaScript Implementation**

```js

  for (let i = 0; i < arr.length; i++) {
    if (arr[i] === target) return i;
  }
  return -1;
}
```

## âš¡ 3. Better Approach â€” Binary Search

Binary Search uses the **divide and conquer** technique.
Instead of checking every element, it eliminates half the array in each step.

### **Working Principle**

1. Find the **middle** element
2. If it's equal to the target â†’ return index
3. If target < mid â†’ search the **left half**
4. If target > mid â†’ search the **right half**
5. Repeat until left > right

## ğŸ§  4. Binary Search Code

### **JavaScript Implementation**

```js
function binarySearch(arr, target) {
  let left = 0;
  let right = arr.length - 1;

  while (left <= right) {
    const mid = Math.floor((left + right) / 2);

    if (arr[mid] === target) return mid;
    if (target < arr[mid]) right = mid - 1;
    else left = mid + 1;
  }

  return -1;
}
```

### **Python Implementation**

```python
def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    
    while left <= right:
        mid = (left + right) // 2

        if arr[mid] == target:
            return mid
        elif target < arr[mid]:
            right = mid - 1
        else:
            left = mid + 1

    return -1
```
